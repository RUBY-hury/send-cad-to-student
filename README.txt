摇号抽CAD图 — Vercel 部署指南
================================

本地直接用：
  双击 index.html 即可在浏览器运行（需 cad_images 文件夹在同目录）

部署到 Vercel（免费）：
  1. 在 https://github.com 注册账号（免费）
  2. 新建仓库，把整个文件夹上传到 GitHub
  3. 打开 https://vercel.com → 用 GitHub 登录 → Import 仓库
  4. 点 Deploy，1分钟后获得链接 https://xxx.vercel.app
  5. 把链接发到 QQ 群即可

注意：
  · cad_images/ 文件夹里有 1.dwg ~ 100.dwg 是占位文件，请替换为真实图纸
  · 学生摇号后浏览器自动下载图纸，记录保存在浏览器本地
  · 同一邮箱不能重复摇号
  · 学生可点击「导出Excel」下载自己的摇号记录
